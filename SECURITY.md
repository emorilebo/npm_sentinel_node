# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

We take the security of `npm-sentinel` seriously. If you discover a security vulnerability within this project, please follow these steps:

1.  **Do NOT open a public issue.**
2.  Email the details to **Godfrey Lebo** at `emorylebo@gmail.com`.
3.  Include a detailed description of the vulnerability and steps to reproduce it.

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.

### Bounty Program
While we do not currently offer a monetary bug bounty, we are happy to publicly acknowledge your contribution in our release notes (with your permission).
